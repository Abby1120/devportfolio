About Me

My name is Abigail and I am a front-end developer working to become a full-stack developer. In 2016 I had an idea for a mobile application that I wanted to build so I hit the tutorials and learned the basics for development.

"Functions are like recipes," states one of my coding mentors. Thinking beyond that, I like to think of being a developer like being a Chef, we don't write the code for ourselves, we are writing it to make the user's lives easier, to help them solve a problem. I'm not making the food for myself I'm providing it for others. This is my process for building the user interface.

My goal is to advance my knowledge to be able to work with several development stacks. For now, I'm learning the MEAN stack and working comfortably with ES6 features. Eliminating JavaScript where CSS can do the job, learning to create modular components, and being able to build useful applications are what drives me to advance my knowledge in programming.

I am actively seeking a Developer Role and would appreciate the opportunity to see how I can contribute to your team.
